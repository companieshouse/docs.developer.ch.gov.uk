How to create an application

Creating an application

To create an application:

  1. Go to create an application
  2. Provide a name and description for the application
  3. Choose whether you want a test application for use against our sandbox environment or a live application for use in production
  4. Provide optional links to the privacy policy URL and terms and conditions URL of your service
  5. Select Create.
  6. A new application will be created. You can then create API clients for your application.

API clients created for to your aplication can then be used to interact with the Companies House API and OAuth 2.0 service. There are 3 types of API client: API key, stream key and oauth web client.

Creating an API client

To create an API client:

  1. Go to application overview
  2. Select the application youâ€™d like to create a key for, or select create an application to create a new one.
  3. Select create new key at the bottom of the page.
  4. Provide a name for the key and choose the client type.
  5. Select Create.
  6. Unique API client credentials will be generated.

API or stream keys can be sent with most GET requests for a public resource served by the Companies House API while oauth web client id and secret combinations can be used to interact with the Companies House OAuth 2.0 service.

API authentication details how to use each type of API client with the Companies House API and OAuth 2.0 services.
